# What Will This Project Provide?
- All CAD Files created for this project
- All code created for this project
- Bill of materials including everything you should purchase / 3D print to create this project
- Step-by-step instructions on how to assemble yourself

# Why This Design?
I picked this design because the specifications required to make an arm with this amount of 2-bars would make for a great challenge and learning opportunity.
